# Food and Beverage Management System

Selamat datang di Aplikasi Fms! Dalam repo ini terdapat 3 Aplikasi
- BackOffice
- Client
- Warehouse

## Package yang Digunakan

- Vite ReactJs
- TRPC
- Drizzle orm
- clsx
- ts-patterns
- Zod
- PostgreSql
- Typescript
- Express

## Getting Started

Ikuti langkah-langkah berikut untuk memulai menggunakan aplikasi ini:

1. **Clone repository:**

   ```
   git clone git@github.com:maulanasdqn/fnb-management.git
   ```

2. **Akses direktori repository:**

   ```
   cd fnb-management
   ```

3. **Install Package:**

   ```
   pnpm install
   ```

## Jalankan pada server dev local:
  **Menjalankan service api :**
  
   ```
   pnpm api:dev
   ```
   Aplikasi akan berjalan di http://localhost:3000.
   
  **Menjalankan modul client :**
  
   ```
   pnpm client:dev
   ```
**Menjalankan modul backoffice :**

   ```
   pnpm backoffice:dev
   ```
**Menjalankan modul warehouse :**

   ```
   pnpm warehouse:dev
   ```
   Aplikasi akan berjalan di http://localhost:4200.
